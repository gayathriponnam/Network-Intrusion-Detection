# Network Intrusion Detection Using Machine Learning

## IBM SkillsBuild AICTE Internship Project

### Project Overview
This project focuses on detecting malicious network activities using Machine Learning. It is developed using **IBM watsonx.ai Studio** and **AutoAI** to automatically build and evaluate machine learning models for identifying network intrusions. The system helps improve cybersecurity by accurately classifying network traffic as normal or malicious.

---

## Problem Statement

Modern computer networks are constantly exposed to cyber threats such as unauthorized access, malware, and Denial-of-Service (DoS) attacks. Traditional intrusion detection methods often fail to detect evolving attacks efficiently. This project aims to develop a machine learning-based Network Intrusion Detection System that accurately classifies network traffic as normal or malicious, enabling faster and more reliable cyber threat detection.

---

## Proposed Solution

- Developed an AI-powered Network Intrusion Detection System using **IBM watsonx.ai Studio** and **AutoAI**.
- Uploaded network traffic data to IBM watsonx.ai for analysis.
- AutoAI automatically performed data preprocessing, feature engineering, and hyperparameter optimization.
- Multiple machine learning models were trained and evaluated to identify the best-performing model.
- The selected model classifies network traffic as **Normal** or **Cyber Attack**, enabling fast and accurate intrusion detection.

## Dataset

The project uses a Network Intrusion Detection dataset containing **40 network traffic features** such as duration, service, flag, source bytes, destination bytes, connection statistics, error rates, login information, and host-based traffic features. These features are used to classify network traffic as either **Normal** or **Malicious**.

---

## Machine Learning Model

- **Model:** Decision Tree Classifier
- **Accuracy:** **99.9%**
